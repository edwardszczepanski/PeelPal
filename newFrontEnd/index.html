About the Enli Heatlh Intelligence Internship
Formerly the Kryptiq Corporation, Enli is a healthcare software company that specializes in health management systems. Much of their products revolve around Electronic Medical Records and data integration.
I worked 300 hours in total for the firm as a full-time intern during the Summer of 2013. My role as a software engineer was to create a new Windows mobile application with the mentorship of a Senior Software developer.
In a short amount of time I created a Windows mobile application that was intended for healthcare patients. It would allow people to better interact with their care professionals and help expand Kryptiq's expansion onto mobile platforms.

Learning Curve
When I applied to this internship I was told I was going to create a web application using JavaScript and HTML. For weeks leading up to the first I day I practiced these languages by developing small web apps. On my first day on the job my mentor informed me that plans had changed and I would be assigned to a new project. My task for the Summer would be to build a fully-fledged Windows Application using C# and XAML using the .NET framework. At that point I had no prior experiences in any of those technologies. In the following weeks I spent countless hours, on the job and off, learning the new programming environment and made real progress on my application. I would have meetings with my mentor and with the general company where I would take notes making sure to absorb as much information as possible.

Development
Most of my work day involved programming and implementing new features into the application. Through this internship I learned about the standards that industry has for software development. The first invaluable skill was the ability to work with version control systems. This picture shows a snapshot of the commits for this project with a diagram for all the branching. I used Mercurial and TortoiseHG to accomplish the version control for this application. Another great thing that I learned was the use of debugging mode to help find the issues in code. In addition, I implemented unit testing to ensure that my application was manipulating data correctly. I had code review sessions with my mentor to ensure that my code was implementing best practices such as proper naming conventions, modularity and readability.

The App
After many revisions and a lot of hard work I was finally able to deploy my mobile application in a closed environment. Being able to work with my functional application on a separate platform was an extremely gratifying experience. The embedded video embedded shows the app function.
Features:
• Review and Send Messages from Care Professionals
• Receive attachments such as Medical Charts
• Review and Cancel Appointments
• Review medications
• Bing Maps API Integration

Presentation
After I had completed my internship I created a presentation of my experience working at the Kryptiq Corporation. Much of my material can be found on this page except I want further in depth into the earlier versions of my application. This speech was given to the entire Kryptiq Corporation which was fantastic as I was able to communicate my work to others and practice my presentation skills. I gave a very similar presentation at the Apprenticeships for Science and Engineering Symposium where I created a trifold board that highlighted my work. This experience solidified my interest in software and has directed my career to where it is today.

  • Date: Summer 2013
